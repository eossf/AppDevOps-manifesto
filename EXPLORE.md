## Round 5 - Target Segment: "No Dev" Companies

**The Problem**

Companies that are 100% SaaS (no in-house dev team) don't fit the model:

```
Pillars that require Dev:
• Coding with AI
• DevOps with AI
• Write features for Agents

Pillars that work without Dev:
• LLM fundamentals
• Agent concepts (for PM oversight)
```

**Three company types identified:**

| Type | Description | Can we serve? |
|------|-------------|---------------|
| Pure SaaS | Only 3rd party tools, no devs | No - different model needed |
| Outsourced Dev | Has product, dev is external agency | Maybe - train PM to manage outsourced AI dev |
| Contractor Dev | 1-2 contractors instead of FTE | Yes - could benefit from AI factory training |

**Conclusion**

Companies with NO developers are likely NOT in the Ideal Customer Profile (ICP). The model requires Dev as an actor (guardrails, quality, architecture).

**Exception**: If the PM can be trained to manage outsourced AI development, there may be value.
